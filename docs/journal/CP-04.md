## Task log

### Task: CP-4 — Handicap & SG stance

- **Status:** Done
- **Date completed:** 2026-05-22
- **Recommendation:** BHP from lowest differentials × 0.96 (18-hole only); SG via par/putts/GIR/FIR heuristic buckets
- **Why:** Coaching-grade trends without GHIN or tour benchmarks
- **Alternatives:** Full WHS with course rating/slope — deferred; tour SG — licensing
- **Risks:** User compares to 18Birdies — UI disclaimer required
- **Next action:** CP-5 rule thresholds; CP-9 pytest from §9 fixtures
- **Evidence / links:** `docs/specs/04-metrics-engine.md`, ADR-005

### Task: 4.1 — Metric formulas

- **Status:** Done
- **Date completed:** 2026-05-22
- **Recommendation:** 15 `metric_key` values across windows; hole-weighted FIR/GIR
- **Why:** Covers PRD scoring leaks, trends, and P1 SG/volatility
- **Alternatives:** Fewer keys, compute on demand — rejected for dashboard latency
- **Risks:** SG proxy arbitrary — document allocation percentages
- **Next action:** Map keys to CP-5 rule conditions
- **Evidence / links:** Spec §4

### Task: 4.2 — Edge cases

- **Why:** Matches CP-3 schema and import reality
- **Alternatives:** Double 9-hole scores for avg — rejected as misleading
